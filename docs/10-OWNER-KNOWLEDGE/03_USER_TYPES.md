## UNIFIED_REGISTRATION_PRINCIPLE

> **Verification Status:** `OWNER_DEFINED`
> **Date:** 2026-06-09

The following workflows should share the same core registration fields:

- New Customer Registration
- New User Registration
- New Employee Creation
- New Sales Representative Creation

Additional role-specific fields may be added separately.

Core registration information should remain unified.
